# 小游戏中心

一个基于 Vue 3 + TypeScript + Vite 开发的小游戏集合。

## 游戏列表

### 谁是卧底
- 支持3-8人游戏
- 每局游戏1名卧底
- 查看身份和投票环节
- 实时显示游戏进度 

### 五子棋
- 经典的双人对战模式
- 黑白棋轮流落子
- 实时计时功能
- 胜负判定提示

### 扫雷
- 三种难度可选
- 支持右键标记
- 移动端长按标记
- 实时显示剩余地雷数

### 数字华容道
- 3x3 到 5x5 可选
- 计步和计时功能
- 移动数字还原图案
- 支持重新开始

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Vant UI
- SCSS

## 功能特点

- 响应式设计，支持移动端

## 开发环境要求
- Node.js >= 16
- npm >= 8

## 部署说明
1. 执行构建命令生成静态文件
2. 将 dist 目录下的文件部署到服务器
3. 配置服务器支持 history 模式路由
4. 设置适当的缓存策略
